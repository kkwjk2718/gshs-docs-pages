# 시작하기

GSHS.app은 경남과학고등학교 학생 생활에 필요한 정보를 한곳에서 빠르게 확인할 수 있도록 만든 서비스입니다.
자주 들어가는 학교 정보와 도구를 더 짧은 동선으로 찾을 수 있게 구성되어 있습니다.

## 어떤 정보를 볼 수 있나요?

- 학교 공지사항
- 오늘과 내일 급식
- 학사일정 달력
- 교내 사이트와 링크모음
- 학생용 도구 모음
- 기상곡 신청과 관련 알림

## 어디에서 접속하나요?

- 서비스 메인: `https://gshs.app`
- 테스트 서버: `https://test.gshs.app`
- 공개 문서: `https://kkwjk2718.github.io/gshs-docs-pages/`

## 첫 화면은 어떻게 구성되어 있나요?

- 홈 화면: 공지, 급식, 일정, 바로 진입할 수 있는 핵심 메뉴 요약
- 메뉴 화면: 기능별 상세 페이지 이동
- 내 정보 화면: 계정 정보와 개인화 화면
- 관리자/방송부 화면: 권한이 있는 계정만 접근 가능

## 처음 사용할 때 추천 순서

1. 홈 화면에서 공지, 급식, 일정 요약을 먼저 확인합니다.
2. 필요한 정보가 있으면 해당 카드나 메뉴로 이동합니다.
3. 계정이 있다면 로그인해서 개인화 정보와 알림을 확인합니다.
4. 오류나 이상 현상이 보이면 서비스 내 오류 신고 기능을 사용합니다.
