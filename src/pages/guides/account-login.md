# 로그인과 계정

로그인을 하면 개인화 홈 정보, 알림, 권한 기반 페이지 접근 기능을 사용할 수 있습니다.

## 로그인은 어디에서 하나요?

- 상단 헤더의 `로그인` 버튼
- 또는 `https://gshs.app/login`

## 로그인 후 무엇이 달라지나요?

- 홈 화면에 개인화 정보가 표시됩니다.
- 읽지 않은 알림을 확인할 수 있습니다.
- 권한이 있는 경우 방송부 스튜디오나 관리자 페이지에 접근할 수 있습니다.

## 내 정보 페이지에서는 무엇을 할 수 있나요?

- 기본 사용자 정보 확인
- 모바일 화면에서 로그아웃
- 계정 상태와 관련된 정보 확인

## 로그아웃은 어떻게 하나요?

- 데스크톱: 상단 사용자 메뉴에서 로그아웃
- 모바일: `/me` 페이지 안의 로그아웃 버튼

## 로그인 문제가 생기면?

- 아이디와 비밀번호를 다시 확인합니다.
- 계속 문제가 생기면 운영자에게 문의합니다.
