# 시작하기

GSHS.app은 경남과학고등학교 학생 생활에 필요한 정보를 한곳에 모아 둔 웹 포털입니다. 자주
들어가는 학교 정보와 도구를 더 빠르게 찾을 수 있도록 구성되어 있습니다.

## 어떤 정보를 볼 수 있나요?

- 학교 공지사항
- 오늘과 내일 급식
- 학사일정 달력
- 교내 사이트 바로가기
- 학생용 도구 모음
- 기상곡 신청과 방송부/관리자 기능

## 어디에서 접속하나요?

- 서비스 메인: `https://gshs.app`
- 테스트 서버: `https://test.gshs.app`
- 공개 문서: `https://docs.gshs.app`

## 첫 화면은 어떻게 구성되어 있나요?

- 상단 헤더: 주요 이동, 로그인, 알림 접근
- 홈 화면: 공지, 급식, 일정, 바로가기 요약
- 메뉴/도구 화면: 상세 기능 진입
- 관리자/방송부 화면: 권한이 있는 계정만 접근 가능

## 처음 사용할 때 추천 순서

1. 홈 화면에서 공지, 급식, 일정 요약을 확인합니다.
2. 더 자세한 내용이 필요하면 해당 카드나 메뉴로 이동합니다.
3. 계정이 있다면 로그인해서 개인화 정보와 알림을 확인합니다.
4. 이상한 점을 발견하면 서비스 내부의 오류 신고 기능을 사용합니다.
